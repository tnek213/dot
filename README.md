# Dot

My personal dotfiles and configuration files.

These files are tailored to my own setup and workflow. I publish them mostly for myself, but share them openly in case they help, inspire, or give ideas to someone else.

## Chezmoi

I use [chezmoi](https://www.chezmoi.io/) to manage and synchronize these files across multiple machines.

## Contributing

This repository is intentionally personal and opinionated. I’m generally not looking to expand it to support broader use cases.
